# Data Format — DOM_MAX

## JSON
- Кодировка: UTF-8
- Отступ: 2 пробела
- Ключи: snake_case
- Даты: ISO 8601 (2026-03-25T14:30:00+03:00)

## Markdown отчёты
- YAML frontmatter с датой и автором
- H1 = название отчёта, H2 = разделы
- Таблицы в GFM-формате

## Аудит-отчёты (формат)
```
# Audit Report: [область] — v[версия]
Дата: YYYY-MM-DD
Метод: automated / manual / automated + manual
Раунд: N

| Проверено | Pass | Warning | Error |
|-----------|------|---------|-------|

AUD-NNNN — [ERROR/WARNING]
- Файл: path:line
- Сейчас: что происходит
- Ожидается: что должно быть (ссылка на L0)
- Каскад: затронутые файлы
- Действие: конкретный фикс
```

## Pydantic models = L0
Все модели данных определяются через Pydantic.
Изменение модели = обновление тестов + миграция БД.
