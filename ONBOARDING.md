## Что такое DOM_MAX

Бот-диспетчер для управляющих компаний (УК) в домовых чатах мессенджера Max.
ФЗ-529 обязывает УК вести чаты — мы делаем инструмент автоматизации внутри этих чатов.

Подробнее: `CLAUDE.md` (читается Claude Code автоматически), бизнес-план в отдельном документе.

---

## Как работаем

- **Репозиторий:** GitHub (приватный), оба коммитим в `main`
- **AI-оператор:** Claude Code (CLI) — читает `CLAUDE.md` и `.claude/rules/`
- **Стек:** Python 3.11, FastAPI, PostgreSQL, Redis, Max Bot API
- **Система качества:** Layer Trust Architecture (L0→L4), подробнее в `.claude/rules/quality-gates.md`

## Быстрый старт (после ответов)

```bash
# 1. Клонировать репо
git clone https://github.com/<аккаунт>/dom-max.git
cd dom-max

# 2. Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Скопировать конфиг
cp .env.example .env
# Заполнить .env своими значениями

# 5. Запустить Claude Code
claude
```
